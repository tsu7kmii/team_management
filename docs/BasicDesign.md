# 基本設計書

## システム概要
チーム開発での進捗管理Webアプリケーションです。  
チームメンバーは各自アカウントを作成し、進捗を追加・管理します。


## 背景・目的
 - 現在、teamsで共有しているExcelファイルで、リーダー一人が書き込むを状況を改善したい
 - 入力時、項目の追加、日付の入力など工数が多い
 - 確認まで時間がかかる(teamsを起動->チームを選択->ファイルを選択->閲覧)
 - スマホでも確認出来るが書き込めない
 - メンバーに認知できていない、共同編集に難がある
 - フレームワークの使用練習
 - MVCモデルでの開発練習
 - 可読性、保守性のあるコードの練習
